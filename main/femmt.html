class Winding

A winding defines a conductor which is wound around a magnetic component such as transformer or inductance. The winding is defined by its conductor and the way it is placed in the magnetic component. To allow different arrangements of the conductors in several winding windows (hexagonal or square packing, interleaved, …) in this class only the conductor parameters are specified. Then, by calling class:Winding in class:VirtualWindingWindow the arrangement of the conductors is specified.

update(method: Optional[str] = None, n_air_gaps: Optional[int] = None, position_tag: Optional[List[float]] = None, air_gap_position: Optional[List[float]] = None, air_gap_h: Optional[List[float]] = None, **kwargs) → None

Updates the air gap structure.

  • Strict keyword usage!

  • All dimensions are in meters

  • all parameters are in lists!

  • first chose the method, second, transfer parameters:

  • Method overview:

    • “center”: ONE air gap exactly in core’s middle

    • “percent”: Easy way to split air gaps over the inner/outer leg

    • “manually”: Place air gaps manually

  • “self.midpoints” is a list with [position_tag, air_gap_position, air_gap_h, c_air_gap]

  • c_air_gap: mesh accuracy factor

  • “EI 2D axi”: position_tag = 0 # ‘-1’: left leg | ‘0’: center leg | ‘1’: right leg

Parameters

  • n_air_gaps (int) – number of air gaps

  • position_tag (float) – specifies the gapped “leg”

  • air_gap_h (float) – List of air gap high, list length depending on total air gap count

  • air_gap_position (float) – specifies the coordinate of the air gap’s center point along the specified leg

  • method (str) – “center”, “percent”, “manually”

Returns

None

Return type

None

Example

>>> import femmt as fmt
>>> geo = fmt.MagneticComponent(component_type='inductor')
>>> geo.air_gaps.update(method="center", n_air_gaps=1, air_gap_h=[0.002]) # 'center': single air gap in the middle
>>> geo.air_gaps.update(method="percent", n_air_gaps=2, position_tag=[0, 0], air_gap_h=[0.003, 0.001], air_gap_position=[10, 80]) # 'percent': Place air gaps manually using percentages
>>> geo.air_gaps.update(method="manually", n_air_gaps=2, position_tag=[0, 0], air_gap_h=[0.003, 0.001], air_gap_position=[0.000, 0.003]) # manually

update_conductors(n_turns: Optional[List] = None, parallel: Optional[List] = None, conductor_type: Optional[List] = None, winding: Optional[List] = None, scheme: Optional[List] = None, conductor_radii: Optional[List] = None, litz_para_type: Optional[List] = None, ff: Optional[List] = None, strands_numbers: Optional[List] = None, strand_radii: Optional[List] = None, thickness: Optional[List] = None, wrap_para: Optional[List] = None, cond_cond_isolation: Optional[List] = None, core_cond_isolation: Optional[List] = None, conductivity_sigma: Optional[List] = None) → None

This Method allows the user to easily update/initialize the Windings in terms of their conductors and arrangement.

Note: set all parameters in a list!

Parameters

  • parallel (list[int]) – Number of parallel turns

  • wrap_para (List) –

  • ff (List) – fill-factor, values between [0….1]

  • winding (List) –

    • “interleaved”

    • ””

  • n_turns (List) – Number of turns in a list [[n_primary], [n_secondary]].

  • winding – Sets the mode how to insert windings in a virtual winding window - “interleaved”: interleaves primary and secondary windings in a single virtual winding window - “primary”: the primary winding in a single virtual winding window - “secondary”: the primary winding in a single virtual winding window

  • scheme (List) –

    if winding != “interleaved” (e.g. “primary” or “secondary”)

    • ”hexa”: hexagonal turn scheme

    • ”square”: square turn scheme using the full winding window height (most common use case)

    • ”square_full_width”: square turn scheme using the full winding window width

    if winding == “interleaved” (works only for transformer, not for inductor!)

    • ”horizontal”: horizontal winding interleaving

    • ”vertical”: vertical winding interleaving

    • ”bifilar”: bifilar winding

    • ”blockwise”: two windings in ONE virtual winding window

  • conductor_type (List) –

    • “stacked” # Vertical packing of conductors

    • ”full” # One massive Conductor in each window

    • ”foil” # Horizontal packing of conductors

    • ”solid” # Massive wires

    • ”litz” # Litz wires

  • litz_para_type – 4 litz parameters, one degree of freedom (3 necessary), e.g. [litz_para_type_primary, litz_para_type_secondary] - “implicit_litz_radius”: needs conductor_radii, ff, strands_numbers - “implicit_ff”: needs conductor_radii, strand_radii, strands_numbers - “implicit_strands_number”: needs conductor_radii, ff, strand_radii

  • strand_radii (List) – radius for a single strand-wire in a litz wire, e.g. [strand_radii_primary, strand_radii_secondary]

  • strands_numbers (List) – Number of single strands in a litz wire, e.g. [strands_numbers_primary, strands_numbers_secondary]

  • conductor_radii (List) – Conductor radii of the litz or solid wire, e.g. [conductor_radii_primary, conductor_radii_secondary]

  • ff – fill-factor, values between [0….1]

  • thickness (List) – foil thickness

  • wrap_para –

  • cond_cond_isolation (List) – Isolation between windings and windings, e.g. [primary2primary, secondary2secondary, primary2secondary]

  • core_cond_isolation (List) – Isolation between windings and the core, e.g. [windings2top_core, windings2bot_core, windings2left_core, windings2right_core],

  • conductivity_sigma (List) – electrical conductivity (sigma) of conductor, e.g. [cond_sigma_primary, cond_sigma_secondary]

Returns

None

Return type

None

>>> geo = fmt.MagneticComponent(component_type="Inductor")
